Linux

Bow Simulator for Linux is either available as a deb package for debian based distributions or as a self-contained AppImage for other distributions. The AppImage only has to be downloaded and made executable (chmod a+x bow_simulator*.AppImage), there is no installation required.

bow_simulator-0.5-linux64.deb

bow_simulator-0.5-linux64.AppImage
